# Phoenix Crypto Checkout

A static crypto payment request app built for GitHub Pages.

## What it does
- Generates a clean payment request message
- Shows the wallet address clearly
- Renders a QR code for quick payments
- Lets you copy the wallet or full payment message

## Deploy with GitHub Pages
1. Push this repo to GitHub
2. Open repository settings
3. Go to **Pages**
4. Set source to **Deploy from a branch**
5. Choose branch **main** and folder **/docs**
6. Save

GitHub Pages will publish the app from the `docs/` folder.
