# Same-Night Money Plan

## Goal
Get payment-ready and sales-ready tonight.

## Steps
1. Use the lead list building offer as the first service.
2. Use the pricing in `../pricing.md`.
3. Use the outreach messages in `../outreach/`.
4. Build one sample lead sheet with 10–20 example rows.
5. Identify one niche to target first.
6. Prepare a payment request using the root payment files.
7. Start outreach once approved.

## Best first niches
- small marketing agencies
- recruiters
- local service businesses
- coaches/consultants

## Rule
Do not overcomplicate. Get first payment first.
